## Alumni Management System

A web-based platform for managing and organizing alumni data in a centralized system. The application allows staff members to add, search, update, and delete alumni records easily through a simple and user-friendly interface.

## Features

* Staff Login and Signup
* Add Alumni Details
* Search Alumni Records
* Delete Alumni Data
* Clean and Responsive Interface

## Technologies Used

* HTML
* CSS
* JavaScript
